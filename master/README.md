# Master Workspace

`master/` 是计划、冻结稿和阶段回顾区，不是正式文档资产区。

这里应该放：

- 当前计划
- 路线图
- 主体冻结稿
- 设计草案
- 阶段回顾

这里不应该放：

- 课程原始要求
- 调研原始资料
- 操作手册

这些内容分别放在：

- `docs/course_requirements/`
- `docs/research/`
- `docs/operations/`

## 当前目录约定

- `master/plans/`
  - 共享计划、职责定义、冻结稿、阶段回顾

- `master/drafts/`
  - 本地临时草稿
  - 默认被 `.gitignore` 忽略

## 使用原则

共享且已经准备讨论或执行的内容，放进 `master/plans/`。  
个人临时整理、AI 中间推演、未冻结笔记，放进 `master/drafts/`。
