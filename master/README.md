# Master Workspace

`master/` 是项目的计划与草稿区，不是正式文档资产区。

这里应该放：

- 当前计划
- 路线图
- 规范草案
- 模块设计草案
- 阶段性讨论稿

这里不应该放：

- 课程原始要求
- 调研资料原件
- 操作手册

这些内容应分别放在：

- `docs/course_requirements/`
- `docs/research/`
- `docs/operations/`

## 目录约定

- `master/plans/`
  - 共享计划、路线图、职责定义、模块目标

- `master/drafts/`
  - 本地临时草稿
  - 默认被 `.gitignore` 忽略

## 当前建议

共享且要讨论的内容，尽量放进 `master/plans/`。  
个人临时整理、AI 迭代中间稿、未冻结笔记，放进 `master/drafts/`。
